# Wear OS Screens

## Navigation

### Touch Navigation
- Swipe left/right to navigate between screens
- Tap buttons to interact

### Rotary Navigation (Galaxy Watch 6)
- Rotate bezel clockwise → Next screen
- Rotate bezel counter-clockwise → Previous screen
- Works on all 4 dashboard screens

## Screens

1. **Home** - Overview with quick stats
2. **Heart Rate** - Real-time heart rate monitoring
3. **Steps** - Daily step count and goal progress
4. **Workout** - Quick workout start

## Ambient Mode

When the watch enters ambient mode (always-on display):
- UI switches to monochromatic colors (battery saving)
- Animations stop
- Simplified layouts
- Essential information only

## Best Practices Applied

Based on Very Good Ventures' Wear OS guide:

- ✅ Material 3 with compact visual density
- ✅ Dark theme optimized for OLED
- ✅ Rotary input support (bezel/crown)
- ✅ Ambient mode with theme adaptation
- ✅ Transparent background for round screens
- ✅ One-finger UI design
- ✅ Self-contained views (minimal scrolling)
- ✅ Page indicators for navigation feedback
